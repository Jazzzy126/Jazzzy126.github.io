## 启动服务器

```BASH
$ hexo server
```

## 创建页面

```BASH
$ hexo new [layout] <title>
```

| 布局      | 路径 |
| ----------- | ----------- |
| post      | source/_posts       |
| page   | source        |
| draft   | source/_drafts        |

## 部署网站

```

```BASH
$ hexo deploy
```

## 清除缓存

```

```BASH
$ hexo clean
```
